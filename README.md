# OldCombatMechanics
1.7/1.8-Style PvP for Modern Minecraft Servers

## Testing Status

Old Combat Mechanics is currently released as a public beta.

The plugin has been built successfully and tested to load on a modern Paper 26.1.2 server. Core startup, configuration loading, and command registration have been verified.

However, this plugin has not yet been fully tested across every real-world server setup, minigame plugin, anticheat, protection plugin, or Paper/Spigot patch version in the supported range.

Because combat mechanics are highly sensitive to server configuration, latency, other plugins, and player expectations, some behavior may need tuning over time.

If you find a bug, unexpected behavior, compatibility issue, or a mechanic that does not feel right, please report it with as much information as possible.

Helpful bug reports include:

* Server software and version, such as Paper 26.1.2 or Spigot 26.2
* Old Combat Mechanics version
* Other PvP, anticheat, minigame, or protection plugins installed
* Your relevant config settings
* Steps to reproduce the issue
* Console errors, if any

Some classic PvP mechanics are approximations using stable public Bukkit/Spigot APIs rather than exact recreations of Minecraft 1.7/1.8 internals.
